- [ ] Clear browser history/cache
- [ ] Set browser zoom to 100%
- [ ] Hide bookmarks bar
- [ ] Use incognito/private mode
- [ ] Disable notifications
- [ ] Close Slack, email, etc.
- [ ] Test microphone audio
- [ ] Practice run-through 2-3 times

### Recording Settings:
- **Resolution:** 1920x1080 (Full HD)
- **Frame Rate:** 60 FPS
- **Audio:** Clear voice, no background noise
- **Screen:** Full screen browser (F11)
- **Cursor:** Visible, smooth movements

### Recording Tools (Choose one):
- OBS Studio (Free, professional)
- Loom (Easy, web-based)
- Camtasia (Paid, full-featured)
- ScreenFlow (Mac)

1. **Record PNG slides** (10-15 seconds each)
2. **Record screen demos** (follow script timing)
3. **Record voice-over** (can be done separately)
4. **Edit together** in video editor

### Editing Tips:
- Add smooth transitions (0.5s fade)
- Background music (low volume, upbeat)
- Zoom in on important UI elements
- Add text overlays for key points
- Color grade for consistency
- Export at 1080p, 60fps
